# Linux Programming 1주차



## IT 동향

 1주차는 이제 막 시작을 한 터라, Linux보다는 현재 IT업계의 동향을 알아보는 시간을 가져보고자 한다.

 현재 대한민국이 새로운 제품이나 기술을 빠르게 빠라가는 **Fast Follower**에서 산업의 변화를 주도하고 새로운 분야를 개척하는 **First Mover**로 변화해감에 따라, 기업이 요구하는 인재상도 점차 변화해가고 있다.



현재 기업에서 추구하는 인재상으로는 **Computational Thinking**으로, 쉽게 해석하자면 컴퓨터를 이용해서 문제를 풀 수 있는 사고능력을 요한다는 것이다. 지금은 AI의 발전으로 AI를 이용해서 문제를 풀 수 있는 사고능력인 **AI Thinking**이라는 단어도 사용되고 있다.





## 진로 방향성

### Web

​	Web developer에서도 Frontend, Backend가 나뉜다. Frontend는 Client에게 서비스를 제공하는 사이트의 개발을 담당하고 Backend에서는 Server에서 기능하는 프로그램의 개발을 담당한다. 이러한 이유로 Client개발자, Server개발자 라고 불리기도 한다.

### App

​	App Developer은 말 그대로 핸드폰 애플리케이션을 개발하는 일을 한다. 대표적으로 Android(JAVA), IOS(Objective C)가 있다

### Game

​	Game programmer은 Game Engine에서의 동작을 프로그래밍하는 일은 하는 것이다. 필자는 현재 Unity를 기반으로 프로그래밍을 하고 있다.

### Security

​	정보 보안 전문가는 갈수록 IT의 중요성이 높아지면서 위험성도 덩달아 증가하고 있는데, 이와 관련된 일을 한다.

### Embedded System

​	Embedded System은 기계나 기타 제어가 필요한 시스템에 대해, 제어를 위한 특정 기능을 수행하는 컴퓨터 시스템으로 보통 Linux나 Rust를 사용하여 개발된다.

### Big Data

​	유의미한 수준으로 축적된 데이터를 가지고 이를 활용하여 데이터를 예측, 생성하는 일을 한다.

### AI(Artificial Intelligence)

​	AI는 축적된 데이터를 학습하여 다음의 행동을 예측하여 수행한다. LLM(Large Language Model)이 유명하다.

### Data Analysis

​	데이터를 수집, 처리, 분석하여 이를 가지고 의사 결정을 지원하는 역할을 한다.

### Server Management

​	서버에 대한 유지보수를 맡는다.





## Linux와 Unix의 차이

### 가격차이

Linux는 무료이고 Unix는 유료이다.

하지만 규모가 커지면 Linux의 Maintenence 비용 증가폭이 더 크기 때문에 규모가 큰 환경의 경우에는 Unix가 경제적이다.
